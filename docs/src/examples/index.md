# 使用示例

文档里的示例按能力拆成了 3 组，先看 `core`，再看 `vue`，最后再看大数据场景。

## Core 示例

这一页聚焦文档模型、路径、节点查询和结构操作。

- 页面：[`/examples/core`](./core.md)
- 适合场景：文档模型、树查询、增删改查、路径定位

## Vue 示例

这一页聚焦 `useJsonDocument()`、`JsonupViewer`、主题和自定义渲染。

- 页面：[`/examples/vue`](./vue.md)
- 适合场景：JSON 查看器、Vue 编辑器、主题切换、自定义节点

## 大数据与性能

这组示例聚焦 `10w+` 节点场景，展示扁平节点、可见行计算和虚拟滚动是如何一起工作的。

- 页面：[`/examples/10w`](./10w.md)
- 配套说明：[`/virtual`](../virtual.md)

## 推荐阅读顺序

1. 先看 [`Core 示例`](./core.md)
2. 再看 [`Vue 示例`](./vue.md)
3. 最后看 [`10w+ 数据`](./10w.md) 和 [`虚拟滚动`](../virtual.md)
